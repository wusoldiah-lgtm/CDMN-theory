# Context-Dependent Measurement Nonlinearity (CDMN)

Author: Iver Patricio Farías Ormazábal  
Date: April 14, 2026  
Location: Santiago, Chile  

---

## Description

This repository contains the original formulation and experimental proposal of the CDMN hypothesis.

The work explores the possibility that quantum measurement outcomes may depend on global measurement context rather than purely local observables.
